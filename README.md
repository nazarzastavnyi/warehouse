# warehouse
***Technologies***
Typescript, Express, Webpack, AWS Dynamodb, Postman<br/>
***Set up project***
```
npm i
npm run build
```
***Start project***<br/>
First you need to set up env (you can see .env.example)
```
PORT=8000
ENV=development
AWS_REGION=
AWS_ACCESS_ID=
AWS_SECRET_KEY=
```
```
npm start
```
